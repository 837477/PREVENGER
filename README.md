# PREVENGER
> AI 실시간 상황 탐지 시스템 (Front-end)

세종대학교 제 6회 AI 인공지능 해커톤 대회에서 개발한 프로젝트입니다.

개발 포지션은 Front-end로, 첫 프론트 작업 프로젝트였습니다. 

아깝게 본선에는 진출하지 못했지만 프론트 개발을 입문 할 수 있는 좋은 기회였습니다.
